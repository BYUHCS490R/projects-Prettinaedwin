Link and Path Examples

Absolute Link

An absolute link includes the full web address. Example: Church Website

Root

The root is the top-level folder of a website. Every path starts from the root, and folders/files inside it create the rest of the path.

Default Document

A default document is the file that loads automatically when you visit a folder, such as index.html.

Absolute vs. Relative Links

Absolute links include the full URL (works anywhere). Relative links point to files in your own site (shorter and easier to move around). You use relative links so your site works no matter where itâ€™s hosted.

Three Rules for Relative Links

  1. Use ./ for something in the same folder.
  2. Use ../ to go up one folder.
  3. Be exact with file/folder names (case matters on most servers).

Do Links Have to Be in NAV?

No. Links can go anywhere (paragraphs, footers, etc.). The <nav> tag is just for main navigation.

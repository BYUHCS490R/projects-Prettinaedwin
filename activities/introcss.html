Explain the difference between inline, embedded, and external CSS.

Inline CSS is Written inside the element (style=""). Quick but messy. Embedded CSS is Written in style tag and inside head. Works only for one page. External CSS is Written in a separate .css file and linked. Best for reusability, clean code, and faster loading.

Why Use Only External CSS? Cleaner HTML it is Easy to maintain.Reusable across pages.Better performance (browser caching)

What is a Selector Used For? A selector tells the browser which elements the CSS rules apply to.
